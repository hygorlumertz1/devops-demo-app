Integração Contínua

Pipeline automatizado que valida o código a cada push, executando lint e testes.

🚀

Entrega Contínua

Deploy automático para o ambiente de produção após a aprovação do pipeline.

🏗️

Infraestrutura como Código

Toda a infraestrutura AWS provisionada via Terraform, versionada no repositório.

Fluxo do Pipeline

  1. Push Código enviado ao GitHub
  2. Lint Validação automática de HTML/CSS
  3. Test Testes automatizados
  4. Build Geração dos artefatos
  5. Deploy Publicação no S3/CloudFront
